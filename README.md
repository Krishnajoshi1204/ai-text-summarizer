# AI-Powered Text Summarizer

This repository contains a Python application for summarizing large texts using AI/NLP techniques.  
It supports both **abstractive** (transformer-based) and **extractive** (LexRank) summarization, batch processing, a Streamlit web UI, and optional ROUGE evaluation.

---

## Quick Description
An AI-powered text summarizer using Hugging Face Transformers, with support for batch article testing, CLI interface, and extensible NLP features.
